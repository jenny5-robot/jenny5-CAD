# Jenny5 robot

A human size, multi purpose robot

## Software needed

OpenSCAD from http://www.openscad.org/

## How to compile

Go to robot folder, open robot.scad with OpenSCAD and press F5 ... and wait for several seconds (or minutes if you have a slow computer).
